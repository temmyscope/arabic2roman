## Convert Arabic Numbers to Roman Numerals

- Requirements: python3

### Run Code
```sh
python3 converter.py
```

### Print Response of Function
```py
print( arabicToRoman(9) )
```

### Run Tests
```sh
python3 -m unittest test.py
```
